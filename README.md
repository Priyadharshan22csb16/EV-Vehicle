# EV Vehicle Trend Analysis Dashboard

## Project Overview
This project analyzes Electric Vehicle (EV) population trends using Python, Pandas, MySQL, SQL, and Power BI.

## Features
- Data Cleaning using Pandas
- Exploratory Data Analysis
- SQL Analytics Queries
- ETL Workflow
- Interactive Power BI Dashboard

## Technologies Used
- Python
- Pandas
- MySQL
- SQL
- Power BI
- Matplotlib

## Dashboard Insights
- EV growth trends
- Top EV brands
- EV type distribution
- Geographic analysis
- Average electric range
